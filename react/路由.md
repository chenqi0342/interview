# react 路由

## React-Router有哪几种路由模式
### BrowserRouter
### 浏览器的路由方式，开发中最常使用
### HashRouter
### 在路径前面加入#号成为一个哈希值，Hash模式的好处是，不会因为刷新找不到对应路径
### MemoryRouter
### 不存储history，所有路由过程保存在内存里。不能进行前进后退，因为地址拦没有发生变化
### NativeRouter
### 经常配合RN使用，多用于移动端
### StaticRouter
### 设置静态路由，需要和后台服务器配合设置，服务端渲染时使用
